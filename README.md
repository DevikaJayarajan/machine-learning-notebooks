# Machine Learning Notebooks

This repository contains machine learning experiments developed during my Master's in Data Science at RMIT University.

The notebooks focus on **predicting customer churn in a telecom dataset** using exploratory data analysis and multiple machine learning models.

---

# Project Overview

Customer churn prediction is an important problem for telecom companies, as it helps identify customers who are likely to leave the service.  
This project explores the **Telco Customer Churn dataset** to understand customer behaviour and build predictive models.

The project is organised into two phases:

---

# Phase 1 – Exploratory Data Analysis (EDA)

Notebook: `Phase1.ipynb`

In this phase, the focus is on understanding the dataset and preparing the data for modelling.

Key steps:

- Loading and exploring the telecom churn dataset
- Handling missing values
- Detecting outliers and duplicate records
- Data sampling and aggregation
- Descriptive statistics
- Data visualization using:
  - Matplotlib
  - Seaborn

The goal of this phase is to identify patterns and relationships that may influence customer churn.

---

# Phase 2 – Predictive Modelling

Notebook: `Phase2.ipynb`

In this phase, several machine learning models are built to predict whether a customer will churn.

Models implemented include:

- Naive Bayes
- Random Forest
- K-Nearest Neighbors (KNN)
- Decision Tree
- Neural Network

Techniques used:

- Feature selection
- Model training and evaluation
- Cross validation
- Hyperparameter tuning
- Model comparison

Performance metrics such as accuracy, confusion matrix, and ROC-AUC are used to evaluate the models.
